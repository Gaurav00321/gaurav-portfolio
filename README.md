## ğŸ“¦ Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Gaurav00321/gaurav-portfolio.git
   cd gaurav-portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   pnpm install
   # or
   yarn install
   ```

3. **Run the development server**
   ```bash

   # or

   ```

   Navigate to [http://localhost:3000](http://localhost:3000)
